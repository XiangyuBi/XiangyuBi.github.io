Exercises in Programming Style 小结

Posted on 2018-12-26

上个月偶然看到Exercises in Programming Style, 看章节和github代码感觉还不错就随手买了一本，半个月大概就翻完了。 顾名思义这本书介绍了各种的编程范式，所有的代码实例都是Python,每个小节用不同的范式来解决统计文本频率的问题。书里涵盖的范围非常之广，从原始的脚本式编程到函数式，oop都有涉及，例子都十分简单。书里的内容大多都是启发式的，顺带提及这种范式在系统中是如何实践的，在过去是如何发展的。

过去的几个月里，我经常觉得自己是因为转专业在很多问题上感到力不从心，无处下手。常常想如何从一个更系统合理的角度去解决一个问题，到最后发现无非是阅读量太小，对大多数材料和问题都只是知其然而不知所以然。过去研究akka，知道什么是actor model，但是为什么要用actor model，除了actor model还有类似解决多线程问题的方案吗。这本书循序渐进的，从历史的却很微观的角度解释了许多类似的问题，细细想来受益匪浅。
